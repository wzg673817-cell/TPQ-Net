# LFM-SEI Multi-SNR I/Q Dataset

The complete experiment-ready LFM I/Q datasets used in the TPQ-Net paper are
available in the GitHub Release:

[Download TPQ-Net Dataset v1.0](https://github.com/wzg673817-cell/TPQ-Net/releases/tag/v1.0.0)

## Dataset contents

The dataset contains six radar emitter classes, with 1,000 samples per emitter and
6,000 samples under each signal condition.

The released signal conditions are:

- Clean
- 30 dB
- 20 dB
- 15 dB
- 10 dB
- 5 dB
- 0 dB
